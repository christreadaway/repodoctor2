Mac Setup Guide

Prerequisites

You need Python 3.12+ and git installed. Most Macs have git pre-installed. To check:

python3 --version    # Need 3.12+
git --version        # Need any version
pip3 --version       # Comes with Python

If Python is missing, install it from python.org or via Homebrew: brew install python@3.12

1. Clone the Repo

cd ~
git clone https://github.com/christreadaway/repodoctor2.git
cd ~/repodoctor2

2. Create Virtual Environment

cd ~/repodoctor2
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

3. Get Your API Keys

GitHub Personal Access Token

Go to GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)

Create a token with the repo scope (full control of private repos).

Anthropic API Key

Go to console.anthropic.com → API Keys → Create Key

This powers the AI project summaries. Costs ~$0.001 per summary using Haiku.

4. Launch the App

cd ~/repodoctor2
source venv/bin/activate
python app.py

The app starts at http://127.0.0.1:5001. On first launch, you'll enter your API keys and set a password.

5. Create a Mac Shortcut (Double-Click to Launch)

A start.command file is included in the repo. Double-click it to launch RepDoctor2 and auto-open the browser.

First-time setup:

chmod +x ~/repodoctor2/start.command

This makes the file executable. You only need to do this once.

Add to your Dock:

  1. Open Finder and go to ~/repodoctor2/
  2. Find start.command
  3. Drag it to the right side of your Dock (past the divider line)
  4. Now you can click it anytime to launch RepDoctor2

Or create a Desktop alias:

ln -s ~/repodoctor2/start.command ~/Desktop/RepDoctor2

6. Keyboard Shortcut (Optional)

To launch RepDoctor2 with a keyboard shortcut like Cmd+Shift+R:

  1. Open Automator (search in Spotlight)
  2. Choose Quick Action
  3. Set "Workflow receives" to no input in any application
  4. Add a Run Shell Script action
  5. Paste this script: 
    ~/repodoctor2/start.command
  6. Save as "Launch RepDoctor2"
  7. Go to System Settings → Keyboard → Keyboard Shortcuts → Services
  8. Find "Launch RepDoctor2" and assign Cmd+Shift+R

Troubleshooting

"Permission denied" when double-clicking start.command

Run chmod +x ~/repodoctor2/start.command in Terminal.

"python3: command not found"

Install Python: brew install python@3.12 or download from python.org.

Port 5001 already in use

Another app is using port 5001. Kill it: lsof -ti:5001 | xargs kill

App opens but can't connect to GitHub

Check your GitHub PAT hasn't expired. Go to Settings in RepDoctor2 and reset credentials.
